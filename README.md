# UMAD-L
Unified Modular Architecture Design-Language

Why do we even write code anymore? Ive always liked UML and people have wanted to make it into a language as its very much possible but the code part has been a massive issue until now with AI.
With AI we can now do the code generation for free but we need a good way to communicate our architecture to the agents and i belive this will be the way. Its solid, easy to understand and gives us more control as programmers what AI will do. Without having to be specific about every detail we can give AI a framework to develop and by logic it should devlop the details for us.

Devlopment will go faster and with a better overview of all the code. It wont be shooting fish in a barrel anymore but a highly effective one.
